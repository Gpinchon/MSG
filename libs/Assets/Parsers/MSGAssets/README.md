
# MSG Assets
## Description
This parser loads MSG assets files with extension `msgassets` and mime type `model/msgAssets+json`.
This file format is used to aggregate multiple assets sources and "embellish" such assets by adding MSG specific things such as fog areas, box projected image based lights...
## Documentation
For documentation on this format please refer to [this page](./doc/specs.md).
The file format follows the JSON specs, a full schema is available [in this folder](./schemas).
